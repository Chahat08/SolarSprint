About and Credits

ABOUT

I made this project for MLH's Jan 2021 Hackathon, Snakes and Hackers. This was the biggest project that I've taken up ever and I'm proud of how it turned out to be.

There is still a lot of scope for improvement and I know what all I could've done better.

For any fellow developers lurking here, I've listed the resources I consulted so that you can benefit from them.
